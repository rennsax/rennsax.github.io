# Rennsax's Blog

My personal blog, powered by [Jekyll][jekyllrb] and the [chirpy][chirpy] theme.

## Bootstrap

Deploy the website locally:

``` shell
bundle install
bundle exec jekyll serve --livereload
```

Development:

``` shell
yarn
yarn build # Build js/css assets.
```
